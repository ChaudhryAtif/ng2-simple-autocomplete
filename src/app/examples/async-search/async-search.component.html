Async search results

This exmample utilizes Github open API.

When search changes, parent component of ng2-simple-autocomplete fetches github repository by search value. Before binding the fetch results to searchResult, it should be processed to match with AutoCompleteItem interface.

If you'd like to show loading spinner on left, you have to set isLoading property from async process. And This exmaple has historyId. So selected item will be remain in history list.
