Async search results

This exmample utilizes Github open API. When search changes, fetch github repositories by search value from `onChangeInput`. Result items should be processed to match with AutoCompleteItem interface. Then reassign it to serachResult.

If you'd like to show loading spinner on left, you have to set isLoading property from async process. And This exmaple has historyId. So selected item will be remain in history list.
